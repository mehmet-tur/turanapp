# Booking State Machine

`PAYMENT_PENDING -> CONFIRMED | CANCELLED_BY_CUSTOMER | CANCELLED_BY_TALENT | READY`

`CONFIRMED -> CANCELLED_BY_CUSTOMER | CANCELLED_BY_TALENT | READY | COMPLETED`

`READY -> IN_PROGRESS | NO_SHOW_CUSTOMER | NO_SHOW_TALENT`

`IN_PROGRESS -> COMPLETED | DISPUTED`

`COMPLETED -> SETTLEMENT_PENDING`

`SETTLEMENT_PENDING -> SETTLED`

`CANCELLED_BY_CUSTOMER -> REFUND_PENDING | REFUNDED`

`CANCELLED_BY_TALENT -> REFUND_PENDING | REFUNDED`
